## Usage

Files in this repo are embedded into the Good Plans Notion workspace via HTMLPreview. To embed a file in Notion, use:
`https://htmlpreview.github.io/?https://raw.githubusercontent.com/goodplansnyc/knowledge/main/FILENAME.html`
